# Setup openWeather

## Créer un compte
1. Se créer un compte (OpenWeather)[https://home.openweathermap.org/]
2. S'abonner au (OneCall api)[https://openweathermap.org/api/one-call-3]
3. Mettre une limite de 1000 call ou moins par jours (Ici)[https://home.openweathermap.org/subscriptions]
Les 1000 premiers calls par jours sont gratuits
4. Ajouter un .env avec la clé d'api
