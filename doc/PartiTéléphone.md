# Parti téléphone
 (Provider)[https://www.twilio.com/]
 
## Avoir un numéro de Trial
1. Se rendre sur le site du provider
2. Se créer un compte
3. Rentrer votre numéro/le numéro que vous aller utiliser pour envoyer vos coordonnées(numéro client)
4. Vérifier son numéro de téléphone
5. Sauvegarder son recovery code
6. Continuer avec le Trial
7. Choisir son type d'utilisation (Student -> Dev -> With minimal code -> other -> sms )
8. Aller dans Messaging dans la sidebard à gauche et ouvir le sous-menu Try it out
9. Claim Trial phone number
10. Voilà


## Ajouter un nouveau numéro personnel 
1. Dans son dashbord aller dans la sidebard de gauche, dans le sous menu manage ouvrir Verified Caller ID
2. Ajouter un nouveau Caller ID avec le bouton en haut à droite
